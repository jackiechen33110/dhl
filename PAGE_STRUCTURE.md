## 登录页

**URL**: `/login`  
**访问权限**: 公开（未登录用户）  
**文件**: `views/login.ejs`

### 页面结构

```
┌─────────────────────────────────────┐
│   DHL 回邮单系统                     │
│   物流回邮单管理平台                 │
├─────────────────────────────────────┤
│                                     │
│  用户名: [________________]          │
│  密码:   [________________]          │
│                                     │
│  [ 记住我 ]  [ 登 录 ]               │
│                                     │
└─────────────────────────────────────┘
```

### 功能说明

| 元素 | 说明 |
|------|------|
| **用户名输入框** | 输入用户名（admin 或 staff） |
| **密码输入框** | 输入密码 |
| **记住我复选框** | 保持登录状态 |
| **登录按钮** | 提交登录请求 |

### 交互流程

1. 用户输入用户名和密码
2. 点击"登录"按钮
3. 系统验证凭证
4. 成功 → 跳转到仪表板
5. 失败 → 显示错误提示

### 默认账户

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |
| staff | staff123 | 员工 |
